## Installation

Use your preferred [NodeJS](https://nodejs.org) package manager to install the CLI globally

```sh
$ npm install --global @visbot/hfs
```

To use the tool without installation, you can use the `npx` command:

```sh
$ npx @visbot/hfs
```
